# Sankalp — Hanuman Sadhana

A spiritual discipline tracker PWA for daily chanting, meditation, pranayama, and focus sessions.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Features

- 📿 **Digital Mala** — Tap-based mantra counter with custom presets
- ⏱️ **Focus Timer** — Concentration sessions with XP rewards
- 🌬️ **Pranayama** — Guided breathing patterns (Box, 4-7-8, Equal)
- 🧘 **Guided Meditations** — Built-in and custom uploaded meditations
- 🏆 **Chant Challenges** — Long-term mantra goals with progress tracking
- 🎵 **Audio Library** — Background ambient tracks with custom uploads
- 📊 **Progress Tracking** — Charts, heatmaps, and session history
- 🎨 **Themes** — Midnight, Dawn, and Temple color schemes
